# Local-SD → kids' UX — topic set

Provenance: assembled via `/user-brainstorm` over 3 rounds (10-topic seed +
2 gap-fill/scope rounds with the operator). 17 topics. Scope narrowed by the
operator to **Play (non-transcription) + Child Kiosk** — the areas that feel
"a little boring." Explicitly out of scope (working well): Kids & toyboxes
management, Settings, Play-Transcription. The underused **Children tab** is an
opportunity (per-child personalization). Seed topic "reward/sticker art" was
dropped by the operator.

Each topic gets one investigation file `NN-<slug>.md` (350-600 words, template
in `plan.md`). Tier = impact on the "make play/kiosk less boring" goal.
